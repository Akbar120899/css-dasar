#Font Styling

    • #Text Styling

    text pada css terkait dengan property pendukungnya seperti warna, indentas, jarak antar kata, jarak antar huruh dll

    a. berbagai property text di dalam css

  • color
  • untuk memberi warnah pada tulisan
  • text-align
  • untuk mengatur format pragraf/teks, seperti rata kiri, rata kanan atau rata kiri dan kanan
  • text-indent
  • memberi indentasi pada pragraf/text atau menggeser tulisan yang pertama
  • text-decoration
  • mengatur dekorasi pada teks atau memberi teks seperti di coret
  • text-transform
  • mengubah jenis huruf menjadi huruf besar, kecil/kapital seperti membuat hurf besar semua pada teks atau sebaliknya
  • letter-spaccing
  • mengatur spasi antar huruf atau tingkat kerapatan/kerenggangan
  • word-spacing
  • mengatur spasi antar kata hampir mirip letter-spacing,

    # Pseudo-Class / Membuat Elemen Html Lbih Kompleks/Syantick

    maksudnya adalah kelas semu yang dimiliki oleh sebuah elemen html, yang membuat kita dapat mendefinisikan style pada "keadaan tertentu" dari elemen tersebut

    pseudo-class ini lebih berkaitan dengan lenk atau dengan elemen yang bisa di klik agar dapat di ubah warnah atau posisi di sat di klik

  • pseudo-class ini terbagi beberapa type diantaranya seperti di bawah
    1. empat type di bawah ini lebih berhubungan dengan link
  • : link
  • style default pada sebuah link (a yang memiliki href)
  • : hover
  • style ketika kursor mouse berada di atas sebuah link/elemen
  • pseudo-class ini aktif/ada perubahan ketika link/elemen di sorot menggunakan mouse (tempatkan mouse ke atas elemennya baru bisa aktif)
  • : active
  • style ketika sebuah link di klik agar membuatnya aktif (keadaan aktif)
  • pseudo-class ini aktif/ada perubahan ketika link/elemen di klik (tempatkan mouse ke atas elemennya kemudian klik baru bisa aktif)
  • : visited
  • style ketika sebuah link sudah pernah di kunjungi sebelumnya (menggunakan browser yang sama) akan menampilkan warnah yang berbeda atau ingin membuatnya tidak ad perbedan
    1. kemudian tiga type ini lebih berhubungan dengan posisi elemen
  • : first-child
  • memilih elemen pertama dari sebuah parent (elemen pembungkusnya)
  • : last-child
  • memilih elemen terakhir dari sebuah parent (elemen pembungkusnya)
  • : nth-child(n)
  • memilih elemen ke-n dari sebuah parent (elemen pembungkusnya)
  • n bisa berarti urutan (1), (2),... atau pola (2n), (3n+2), (4n-1) atau ganjil dan genap (event)&(odd)
